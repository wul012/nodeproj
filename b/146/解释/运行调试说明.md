# Node v146 运行调试说明

本版目标：

```text
新增 real-read smoke release evidence gate。
判断 v145 导入的 smoke evidence 是否能成为 production pass evidence。
```

## 启动决策

本版只启动 Node 自身做 HTTP smoke 和截图。

没有启动 Java / mini-kv。

原因：

```text
v146 是 release evidence gate，不是真实 live smoke；
当前默认 skipped evidence 应被明确判定为 not-production-pass-evidence。
```


## 安全边界

本版 HTTP smoke 和截图只启动 Node 本地服务，且验证后已停止。

运行环境保持：

```text
UPSTREAM_PROBES_ENABLED=false
UPSTREAM_ACTIONS_ENABLED=false
```

本版没有启动、停止、构建、测试或修改 Java / mini-kv。

Java v52 和 mini-kv v61 已经提前完成，本版只读取它们的 tag 和 evidence 语义作为静态 readiness review 输入。

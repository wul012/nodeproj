# Node v143 运行调试说明

本版目标：

```text
新增 live probe real-read smoke archive adapter。
把 smoke harness 的 pass/skipped 结果归一成可归档 evidence record。
```

## 启动决策

本版只启动 Node 自身做 HTTP smoke 和截图。

没有启动 Java / mini-kv。

原因：

```text
v143 是 archive adapter，不是真实上游 smoke；
默认 UPSTREAM_PROBES_ENABLED=false 时，adapter 应归档 skipped evidence，并明确不是 production pass。
```

## 验证结果

本版验证项：

```text
npm run typecheck
npx vitest run test/productionLiveProbeRealReadSmokeArchiveAdapter.test.ts
npm test
npm run build
HTTP smoke
本机 Chrome 截图
```

测试结果：

```text
target test: 1 file / 3 tests passed
full test: 92 files / 319 tests passed
build: passed
```

HTTP smoke 使用安全环境变量：

```text
UPSTREAM_PROBES_ENABLED=false
UPSTREAM_ACTIONS_ENABLED=false
ACCESS_GUARD_ENFORCEMENT_ENABLED=true
```

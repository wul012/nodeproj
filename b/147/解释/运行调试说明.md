# Node v147 运行调试说明

本版目标：

```text
新增 real-read smoke dry-run command package。
把 v144 execution request、v145 result importer、v146 release gate 串成单一 operator package。
```

## 启动决策

本版只启动 Node 自身做 HTTP smoke 和截图。

没有启动 Java / mini-kv。

原因：

```text
v147 是 dry-run package，不是真实 pass evidence capture；
它只打包操作员审阅材料，不打开真实只读窗口。
```

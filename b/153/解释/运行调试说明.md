# v153 运行调试说明：real-read smoke operator runbook

## 本版目标

v153 消费 v152 archive verification，生成真实只读联调前的 operator runbook。

本版重点不是启动 Java / mini-kv，而是把启动前条件写清楚：

```text
operator 手动启动 Java / mini-kv
UPSTREAM_PROBES_ENABLED=true
UPSTREAM_ACTIONS_ENABLED=false
Node 不自动启动上游
只执行 Java GET 和 mini-kv 只读命令
```

## 安全边界

本版没有启动、停止、构建、测试或修改 Java / mini-kv。

Node 临时 smoke / screenshot 服务在验证后停止。

本版明确禁止：

```text
Java replay POST
订单写操作
mini-kv SET / DEL / EXPIRE
UPSTREAM_ACTIONS_ENABLED=true
Node 自动启动 Java / mini-kv
```

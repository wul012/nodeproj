v151 是一次小闭环重构，用来收敛 v144-v150 之间重复增长的工具函数和路由注册模板。

它不新增生产能力，不改变现有 endpoint 路径和响应语义，主要解决：

```text

Markdown render helpers 重复
route JSON / Markdown 注册模板重复

```

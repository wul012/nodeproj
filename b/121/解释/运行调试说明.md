# Node v121 运行调试说明：JWKS verifier fixture rehearsal

本版来源计划：

```text
docs/plans/v119-production-connection-roadmap.md
```

本版目标：

```text
给 v118 的 IdP verifier boundary 增加本地 JWKS / JWT fixture 验签演练。
验证 issuer、audience、expiry、roles、kid。
不访问外部 IdP，不把 fixture token 用作真实授权依据。
```

## 运行环境

本版 smoke 使用安全环境变量：

```text
HOST=127.0.0.1
PORT=4121
UPSTREAM_PROBES_ENABLED=false
UPSTREAM_ACTIONS_ENABLED=false
ORDEROPS_IDP_ISSUER=https://idp.example
ORDEROPS_IDP_AUDIENCE=orderops-node
ORDEROPS_IDP_JWKS_URL=https://idp.example/.well-known/jwks.json
ORDEROPS_IDP_CLOCK_SKEW_SECONDS=90
```

没有启动、停止、构建、测试或修改 Java / mini-kv。

# Node v111 运行调试说明

本次运行调试对象是 `signed auth token contract rehearsal`。

运行边界：

```text
UPSTREAM_PROBES_ENABLED=false
UPSTREAM_ACTIONS_ENABLED=false
ORDEROPS_AUTH_TOKEN_ISSUER=orderops-smoke
ORDEROPS_AUTH_TOKEN_SECRET=smoke-secret-v111
PORT=4121
```

`ORDEROPS_AUTH_TOKEN_SECRET` 只用于本地 HMAC rehearsal。接口只返回 `secretConfigured=true`，不会回显 secret。

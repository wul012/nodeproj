# Node v136 运行调试说明

本版目标：

```text
新增 read-only live probe smoke harness。
在 Java / mini-kv 未启动时不失败，而是记录 graceful skipped evidence。
```

## 验证结果

本版验证项：

```text
npm run typecheck
npx vitest run test/productionLiveProbeSmokeHarness.test.ts
npm test
npm run build
HTTP smoke
本机 Chrome 截图
```

HTTP smoke 使用安全环境变量：

```text
UPSTREAM_PROBES_ENABLED=false
UPSTREAM_ACTIONS_ENABLED=false
ACCESS_GUARD_ENFORCEMENT_ENABLED=true
```

实际 smoke 结果：

```text
profileVersion=production-live-probe-smoke-harness.v1
readyForLiveProbeEvidence=true
probesEnabled=false
skippedProbeCount=5
blockedProbeCount=0
productionBlockerCount=0
markdownStatus=200
```

这说明本版没有访问 Java / mini-kv，也没有做任何写入；五个 planned probes 都按配置记录为 skipped evidence。

# Node v141 运行调试说明

本版目标：

```text
新增 live probe operator handoff checklist。
把 v140 bundle 变成操作员可读的真实只读 smoke 交接清单。
```

## 启动决策

本版没有启动 Java / mini-kv。

原因：

```text
v141 的目标是 handoff checklist，不是实际 live smoke；
真实启动应留到显式 probe window，并由 checklist/switch 先约束边界。
```

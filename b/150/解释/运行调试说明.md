# v150 运行调试说明：production pass evidence archive

## 本版目标

v150 把 v149 verification、v148 capture、v146 release evidence gate 的 digest 链固化成 archive。

默认没有启动 Java / mini-kv，因此本版正常状态是：

```text
archiveState=not-production-pass-evidence-archived
readyForProductionPassEvidenceArchive=true
readyForProductionOperations=false
captureMode=skipped
archivedAsProductionPassEvidence=false
```

这说明 Node 可以归档 skipped 本地证据，但不会把它提升成 production pass。

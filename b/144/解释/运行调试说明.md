# Node v144 运行调试说明

本版目标：

```text
新增 real-read smoke execution request。
把真实只读 smoke 的执行前请求、环境变量、命令 profile 和禁止写动作固化出来。
```

## 启动决策

本版只启动 Node 自身做 HTTP smoke 和截图。

没有启动 Java / mini-kv。

原因：

```text
v144 是 execution request，不是真实 smoke 执行；
它只生成操作员可审阅的命令 profile，不负责启动上游。
```

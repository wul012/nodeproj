# Node v145 运行调试说明

本版目标：

```text
新增 real-read smoke result importer。
把 pass/skipped smoke 输出归一成 schema-backed import envelope。
```

## 启动决策

本版只启动 Node 自身做 HTTP smoke 和截图。

没有启动 Java / mini-kv。

原因：

```text
v145 是 result importer，不是真实 pass evidence；
默认 UPSTREAM_PROBES_ENABLED=false 时，应导入 skipped 样本。
```

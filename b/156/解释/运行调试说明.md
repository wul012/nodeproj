# v156 运行调试说明：read-only window live capture

## 本版目标

v156 消费 v155 readiness packet，并复用现有 live probe smoke harness，执行真实只读窗口 capture 的 Node 侧能力。

本版不会自动启动 Java / mini-kv：

```text
UPSTREAM_PROBES_ENABLED=false 时，只记录 skipped evidence
UPSTREAM_PROBES_ENABLED=true 时，只执行 Java GET 和 mini-kv HEALTH / INFOJSON / STATSJSON
UPSTREAM_ACTIONS_ENABLED 必须保持 false
```

## 安全边界

本版没有启动、停止、构建、测试或修改 Java / mini-kv。

Node 临时 smoke / screenshot 服务在验证后停止。

本版继续禁止：

```text
Java replay POST
订单写操作
mini-kv SET / DEL / EXPIRE
UPSTREAM_ACTIONS_ENABLED=true
Node 自动启动 Java / mini-kv
skipped/mixed evidence 冒充 production pass
```

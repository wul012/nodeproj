# Node v85 说明：Dashboard scenario archive verification panel

## 本版目标

v85 按 `docs/plans/v84-post-scenario-archive-verification-roadmap.md` 推进，只把 v84 已完成的 archive bundle verification 能力展示到 Dashboard。

本版新增的是只读观察能力：

- 展示 `verificationValid`。
- 展示 `archiveBundleDigestValid`、`verificationDigestValid`。
- 展示 `sourcePathCountValid`、`readOnlyStillTrue`、`executionAllowedStillFalse`。
- 提供 archive bundle verification JSON / Markdown 入口。
- 页面加载时自动刷新 verification 面板。

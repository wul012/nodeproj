# 第五十六版运行调试归档说明

本轮版本目标：

```text
Node v56 按最新 plan 指导，把 v53-v55 的只读上游信号整理成 Dashboard 上可扫描的详情面板。
```

本轮只修改 Node，不修改 Java 和 mini-kv，不新增上游写操作。

本轮先写入并遵守规则：

```text

```

规则位置：

```text
docs/plans/README.md
```

本轮按 `docs/plans/v54-post-infojson-roadmap.md` 执行 Node v56。

# 2. 代码改动

## `src/ui/dashboard.ts`

新增三列上游观察详情面板：

```text
Upstream Overview
Java Signals
mini-kv Signals
```

展示内容：

```text
整体状态
probe/action safety flags
Java health
Java business overview
Java failed-event summary
Java replay backlog
Java pending approvals
mini-kv version / protocol
mini-kv live keys
mini-kv WAL / metrics
mini-kv command catalog
mini-kv write/admin/mutating command counts
```

新增前端函数：

```text
renderUpstreamOverview()
refreshUpstreamOverview()
formatBool()
formatNumber()
formatAvailable()
setText()
```

页面加载时会自动刷新 overview；点击 `Upstream Overview` 按钮时，会刷新面板并输出原始 JSON。

## `test/dashboard.test.ts`

新增 Dashboard HTML 结构测试，确认 v56 面板和刷新函数存在。


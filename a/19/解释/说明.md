# OrderOps Node V19 运行调试归档

## 本版目标

V19 新增本地 ops promotion decision ledger：

- `POST /api/v1/ops/promotion-decisions` 固化当前 promotion review。
- `GET /api/v1/ops/promotion-decisions` 列出本地 promotion decisions。
- `GET /api/v1/ops/promotion-decisions/:decisionId` 读取单条 decision。
- 每条 decision 保存 reviewer、note、review snapshot。
- 每条 decision 保存 SHA-256 摘要。
- Dashboard 新增 `Record Decision` 和 `Decisions` 入口。
- 本接口只读取 Node 本地状态，不触碰 Java / mini-kv。

```ts
function digestRecord(record: OpsPromotionDecisionRecord): string {
  return createHash("sha256")
    .update(stableJson({
      sequence: record.sequence,
      createdAt: record.createdAt,
      reviewer: record.reviewer,
      note: record.note,
      outcome: record.outcome,
      readyForPromotion: record.readyForPromotion,
      review: record.review,
    }))
    .digest("hex");
}
```

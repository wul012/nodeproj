# V35 开发运行调试说明

## 本版目标

V35 在 V34 `promotion handoff receipt verification` 后继续收口，新增 `promotion handoff closure`。

目标是把已验证的 receipt 整理成最终交接关闭记录：

```text
handoff package
 -> handoff package verification
 -> handoff certificate
 -> handoff certificate verification
 -> handoff receipt
 -> handoff receipt verification
 -> handoff closure
```

新增接口：

```text
GET /api/v1/ops/promotion-archive/handoff-closure
GET /api/v1/ops/promotion-archive/handoff-closure?format=markdown
```

closure 保存：

```text
closureDigest
receiptDigest
verifiedReceiptDigest
certificateDigest
verifiedCertificateDigest
packageDigest
verifiedPackageDigest
sealDigest
closureItems
```

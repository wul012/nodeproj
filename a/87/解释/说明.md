# Node v87 说明：Release evidence readiness gate

## 本版目标

v87 按 `docs/plans/v86-production-readiness-roadmap.md` 推进，新增只读 release evidence readiness gate。

它消费 v86 的 release evidence index，把证据完整度、digest 覆盖、只读边界、执行禁止边界转成 release gate 结果。

## 本版新增能力

- 新增 release evidence readiness gate service。
- 新增 JSON / Markdown endpoint。
- 输出 `readyForReleaseEvidenceArchive`、blockers、warnings、nextActions。
- gate 消费 v86 index，不重复手写 evidence 结论。
- 明确 ready 只代表“可归档发布证据”，不是上游执行许可。

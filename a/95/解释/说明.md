# Node v95 说明：Production readiness summary index

## 本版目标

v95 按 `docs/plans/v92-production-ops-roadmap.md` 推进，新增生产化 readiness summary index。

它统一汇总 release evidence、CI command profile、workflow verification、deployment safety、rollback runbook、audit store runtime profile，并明确当前不是生产 ready，主要阻塞点在审计持久化。

## 本版新增能力

- 新增 production readiness summary service。
- 新增 JSON / Markdown endpoint：`/api/v1/production/readiness-summary`。
- 汇总 v87-v94 的关键 evidence endpoint。
- 输出每个 section 的 ready/readOnly/executionAllowed/blocker/warning。
- 把 audit store 的生产阻塞项集中暴露。
- 清理过期的 CI workflow 未创建提示，改为验证已存在的 Node evidence workflow。

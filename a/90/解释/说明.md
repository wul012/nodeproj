# Node v90 说明：GitHub Actions evidence workflow draft

## 本版目标

v90 按 `docs/plans/v89-production-hardening-roadmap.md` 推进，基于 v88 CI evidence command profile 新增 GitHub Actions evidence workflow 草案。

## 本版新增能力

- 新增 `.github/workflows/node-evidence.yml`。
- workflow 运行 `npm ci`、`npm run typecheck`、`npm test`、`npm run build`。
- safe smoke 使用 `UPSTREAM_PROBES_ENABLED=false`、`UPSTREAM_ACTIONS_ENABLED=false`。
- smoke 检查 `/health` 和 release evidence readiness gate。
- 不写 secret，不部署，不调用 Java / mini-kv。

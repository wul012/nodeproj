# Node v68：approval execution gate preview

## 来源

本版依照 `docs/plans/v65-post-upstream-evidence-roadmap.md` 推进，并新增 `docs/plans/v68-post-execution-gate-roadmap.md` 作为后续小版本计划。

## 本版完成

- 新增 `operationApprovalExecutionGatePreview` 服务
- 新增执行门禁预览 endpoint：
  - `GET /api/v1/operation-approval-requests/:requestId/execution-gate-preview`
  - `GET /api/v1/operation-approval-requests/:requestId/execution-gate-preview?format=markdown`
- 基于 handoff bundle 判断：
  - request 是否 approved
  - decision 是否 approved
  - handoff bundle 是否 ready
  - evidence verification 是否 valid
  - upstream 是否仍未被触碰
  - Java approval digest 证据是否有效
  - mini-kv command digest / side_effect_count 证据是否有效
- 返回 `previewOnly=true` 与 `executionAllowed=false`

## 本版不做

- 不调用 Java replay POST
- 不执行 mini-kv SET/DEL/EXPIRE
- 不新增真实 execution endpoint
- 不修改 Java 或 mini-kv 项目

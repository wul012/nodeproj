# Node v92 说明：Rollback evidence runbook endpoint

## 本版目标

v92 按 `docs/plans/v89-production-hardening-roadmap.md` 推进，新增只读 rollback evidence runbook endpoint。

它把部署前检查、CI evidence、release gate、恢复步骤整理成可归档报告。

## 本版新增能力

- 新增 rollback runbook service。
- 新增 JSON / Markdown endpoint。
- 汇总 v87-v89 的 evidence endpoint。
- 输出 rollback steps、verification steps、stop conditions。
- 明确 runbook 是只读证据，不执行真实回滚。

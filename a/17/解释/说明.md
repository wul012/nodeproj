# OrderOps Node V17 运行调试归档

## 本版目标

V17 新增本地 ops baseline tracking：

- `GET /api/v1/ops/baseline` 读取当前 baseline 状态。
- `PUT /api/v1/ops/baseline` 把某个 checkpoint 设为 baseline。
- `DELETE /api/v1/ops/baseline` 清除 baseline。
- 最新 checkpoint 与 baseline 不同时，自动复用 V16 checkpoint diff。
- Dashboard 新增 `Set Baseline` 和 `Baseline` 入口。
- 本接口只读取 Node 内存 checkpoint，不触碰 Java / mini-kv。

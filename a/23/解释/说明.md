# OrderOps Node V23 运行调试归档

## 本版目标

V23 新增本地 promotion decision ledger integrity Markdown report：

- `GET /api/v1/ops/promotion-decisions/integrity` 继续输出 JSON。
- `GET /api/v1/ops/promotion-decisions/integrity?format=markdown` 输出 Markdown report。
- Markdown 展示 `valid`、记录数量、sequence 范围、root digest、checks 和每条 chain digest。
- Dashboard 新增 `Integrity Report` 入口。
- 本功能只读取 Node 本地 ledger，不触碰 Java / mini-kv。

### 6. sequence range

文件：`src/services/opsPromotionDecision.ts`

```ts
function sequenceRange(integrity: OpsPromotionDecisionLedgerIntegrity): string {
  if (integrity.oldestSequence === undefined || integrity.newestSequence === undefined) {
    return "empty";
  }

  return `${integrity.oldestSequence}..${integrity.newestSequence}`;
}
```

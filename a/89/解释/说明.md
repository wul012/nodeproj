# Node v89 说明：Deployment safety profile

## 本版目标

v89 按 `docs/plans/v86-production-readiness-roadmap.md` 推进，新增部署安全画像，用来检查当前 Node 运行配置是否适合进入生产级演示环境。

## 本版新增能力

- 新增 deployment safety profile service。
- 新增 JSON / Markdown endpoint。
- 汇总 probes/actions 开关、host/port、日志级别、上游 URL、fixture 路径、rate limit。
- 输出 blockers / warnings / recommendations。
- 明确本版不真实部署、不修改服务器环境、不写入密钥。

# OrderOps Node V13 运行调试归档

## 本版目标

V13 新增本地 ops handoff report：

- 汇总 sources、ops summary、ops readiness。
- 携带最近 audit events、operation intents、dispatches、intent events。
- 支持 JSON 和 Markdown 两种输出。
- Dashboard 新增 `Handoff Report` 只读入口。
- 默认 `UPSTREAM_PROBES_ENABLED=false` 时，Java / mini-kv 仍然显示 `disabled`，不会被探测。

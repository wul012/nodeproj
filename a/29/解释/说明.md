# OrderOps Node V29 运行调试归档

## 本版目标

V29 新增本地 promotion handoff package：

- `GET /api/v1/ops/promotion-archive/handoff-package` 输出 JSON handoff package。
- `GET /api/v1/ops/promotion-archive/handoff-package?format=markdown` 输出 Markdown handoff package report。
- 汇总 V24 archive bundle、V25 manifest、V26 verification、V27 attestation、V28 attestation verification。
- 为五个附件生成 attachment digest、source 和 valid 状态。
- 生成最终 `packageDigest`，覆盖 package name、archive name、state、handoffReady、sealDigest、manifestDigest、verificationDigest、summary、attachments 和 nextActions。
- Dashboard 新增 `Handoff Package` 和 `Handoff Package Report` 入口。
- 本功能只读取 Node 本地内存状态，不触碰 Java / mini-kv。

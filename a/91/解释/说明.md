# Node v91 说明：Persistent audit store adapter

## 本版目标

v91 按 `docs/plans/v89-production-hardening-roadmap.md` 推进，新增可替换的审计持久化适配层，为以后从内存审计迁移到文件/数据库做准备。

## 本版新增能力

- 新增 `AuditStore` interface。
- 新增 `InMemoryAuditStore`，保持默认行为不变。
- 新增 `FileBackedAuditStore` JSONL prototype。
- `AuditLog` 通过 store adapter 读写事件。
- 补测试证明现有审计 API 不回退，并验证 file-backed prototype 能恢复事件。

# Node v86 说明：Scenario release evidence index

## 本版目标

v86 按 `docs/plans/v84-post-scenario-archive-verification-roadmap.md` 收口，新增只读 release evidence index，把四类 scenario evidence 汇总到一个索引：

- scenario matrix
- scenario matrix verification
- scenario archive bundle
- scenario archive bundle verification

同时把新的总目标写入工程方向：后续版本要逐步靠近生产级，不再只做作品级展示。

## 本版新增能力

- 新增 release evidence index service。
- 新增 JSON / Markdown endpoint。
- 每个 evidence 都有 endpoint、digest、valid、readOnly、executionAllowed 摘要。
- index 自身生成 `releaseEvidenceDigest`。
- 明确 `maturityTarget=production-leaning`。
- 明确 `executionAllowed=false`，该索引不是执行许可。

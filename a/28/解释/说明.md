# OrderOps Node V28 运行调试归档

## 本版目标

V28 新增本地 promotion archive attestation verification：

- `GET /api/v1/ops/promotion-archive/attestation/verification` 输出 JSON verification。
- `GET /api/v1/ops/promotion-archive/attestation/verification?format=markdown` 输出 Markdown verification report。
- 复算 V27 `sealDigest`，确认 handoff seal 自洽。
- 复算 V27 `verificationDigest`，确认 attestation 引用的 V26 verification 仍然一致。
- 对比 manifest digest、archive name、state、handoffReady、decision、checks、evidence sources 和 nextActions。
- Dashboard 新增 `Attestation Verification` 和 `Attestation Verification Report` 入口。
- 本功能只读取 Node 本地内存状态，不触碰 Java / mini-kv。

# V32 开发运行调试说明

## 本版目标

V32 在 V31 `promotion handoff certificate` 后面继续加一层复核能力：

```text
handoff package
 -> handoff package verification
 -> handoff certificate
 -> handoff certificate verification
```

新增接口：

```text
GET /api/v1/ops/promotion-archive/handoff-certificate/verification
GET /api/v1/ops/promotion-archive/handoff-certificate/verification?format=markdown
```

这版的目的：不要只相信 V31 证书自己带的 `certificateDigest`，而是重新计算 digest，并检查 certificate 引用的 package、verified package、seal、decision、verification 和 attachments 是否仍然和当前归档链一致。

# OrderOps Node V15 运行调试归档

## 本版目标

V15 新增本地 ops checkpoint ledger：

- `POST /api/v1/ops/checkpoints` 捕获当前 summary/readiness/runbook。
- `GET /api/v1/ops/checkpoints` 列出本地 checkpoint。
- `GET /api/v1/ops/checkpoints/:checkpointId` 读取单条 checkpoint。
- 每条 checkpoint 带本地递增 `sequence`。
- 每条 checkpoint 带 SHA-256 摘要。
- Dashboard 新增 `Create Checkpoint` 和 `Checkpoints` 只读/本地入口。
- 本接口不调用 `OpsSnapshotService.sample()`，不会触碰 Java / mini-kv。

```ts
function digestCheckpoint(checkpoint: OpsCheckpoint): string {
  return createHash("sha256")
    .update(stableJson({
      sequence: checkpoint.sequence,
      createdAt: checkpoint.createdAt,
      actor: checkpoint.actor,
      note: checkpoint.note,
      decision: checkpoint.decision,
      snapshot: checkpoint.snapshot,
    }))
    .digest("hex");
}
```

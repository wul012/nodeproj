# OrderOps Node V25 运行调试归档

## 本版目标

V25 新增本地 promotion archive manifest：

- `GET /api/v1/ops/promotion-archive/manifest` 输出 JSON manifest。
- `GET /api/v1/ops/promotion-archive/manifest?format=markdown` 输出 Markdown manifest report。
- Manifest 汇总 archive summary、latest evidence、ledger integrity 三类 artifact。
- 每个 artifact 都带 SHA-256 digest。
- Manifest 自身也生成 SHA-256 `manifestDigest`。
- Dashboard 新增 `Archive Manifest` 和 `Manifest Report` 入口。
- 本功能只读取 Node 本地 ledger，不触碰 Java / mini-kv。

### 4. 创建 manifest

文件：`src/services/opsPromotionArchiveBundle.ts`

```ts
export function createOpsPromotionArchiveManifest(bundle: OpsPromotionArchiveBundle): OpsPromotionArchiveManifest {
  const artifacts = archiveArtifacts(bundle);
  const manifestPayload = {
    archiveName: bundle.archiveName,
    state: bundle.state,
    summary: bundle.summary,
    artifacts: artifacts.map((artifact) => ({
      name: artifact.name,
      type: artifact.type,
      present: artifact.present,
      digest: artifact.digest.value,
      source: artifact.source,
    })),
    nextActions: bundle.nextActions,
  };
```

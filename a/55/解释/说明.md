# 第五十五版运行调试归档说明

本轮版本目标：

```text
Node v55 接入 Java v37 failed-event summary 和 mini-kv v46 COMMANDSJSON，在 /api/v1/upstreams/overview 输出只读风险摘要。
```

本轮只修改 Node 项目，不修改 Java 和 mini-kv 源码。

---

# 1. 代码改动

## `src/clients/orderPlatformClient.ts`

新增：

```text
OrderPlatformFailedEventSummary
failedEventsSummary()
```

读取：

```text
GET /api/v1/failed-events/summary
```

## `src/clients/miniKvClient.ts`

新增：

```text
MiniKvCommandCatalogEntry
MiniKvCommandsJson
MiniKvCommandsJsonResult
commandsJson()
parseMiniKvCommandsJson()
```

读取：

```text
COMMANDSJSON
```

并允许 raw gateway command 白名单识别：

```text
COMMANDS
COMMANDSJSON
```

注意：真实上游动作仍受 `UPSTREAM_ACTIONS_ENABLED=false` 默认保护。

## `src/services/opsSnapshotService.ts`

新增 fallback probe：

```text
OrderPlatformFailedEventSummaryProbe
MiniKvCommandCatalogProbe
```

Java health 成功后并发读取：

```text
/api/v1/ops/overview
/api/v1/failed-events/summary
```

mini-kv 探测并发读取：

```text
PING
HEALTH
STATSJSON
INFOJSON
COMMANDSJSON
```

## `src/services/upstreamOverview.ts`

Java signals 新增：

```text
failedEventSummaryAvailable
failedEventSummaryLatencyMs
failedEventSummary
failedEventReplayBacklog
failedEventPendingReplayApprovals
failedEventApprovedReplayApprovals
failedEventRejectedReplayApprovals
```

mini-kv signals 新增：

```text

```

# 3. 安全 smoke

环境：

```text
PORT=4195
UPSTREAM_PROBES_ENABLED=false
UPSTREAM_ACTIONS_ENABLED=false
```

结果：

```json
{
  "healthStatus": "ok",
  "overviewState": "disabled",
  "javaState": "disabled",
  "miniKvState": "disabled",
  "failedEventSummaryAvailable": false,
  "commandCatalogAvailable": false,
  "upstreamActionsEnabled": false
}
```

意义：

```text
默认安全配置下 Node 不触碰 Java 或 mini-kv，也不会打开真实写动作。
```

# 4. 真实三项目只读联调

本次启动：

```text
Java v37: 127.0.0.1:18082
mini-kv v46: 127.0.0.1:16381
Node v55: 127.0.0.1:4195
```

结果摘要：

```json
{
  "nodeHealth": "ok",
  "overallState": "online",
  "javaState": "online",
  "javaFailedSummaryAvailable": true,
  "javaReplayBacklog": 0,
  "javaPendingApprovals": 0,
  "miniKvState": "online",
  "miniKvVersion": "0.46.0",
  "miniCommandCatalogAvailable": true,
  "miniWriteCommandCount": 3,
  "miniAdminCommandCount": 4,
  "upstreamActionsEnabled": false
}
```

意义：

```text
Node v55 能只读看见 Java 失败事件治理压力，也能只读看见 mini-kv 命令能力分级。
```

# Node v94 说明：Audit store runtime profile endpoint

## 本版目标

v94 按 `docs/plans/v92-production-ops-roadmap.md` 推进，新增只读 audit store runtime profile endpoint。

它不改变审计存储行为，而是把当前运行时的真实状态讲清楚：默认是 `InMemoryAuditStore`，`FileBackedAuditStore` 仍是 prototype，生产审计还需要 durable store、retention、backup 和访问控制。

## 本版新增能力

- 新增 audit store runtime profile service。
- 新增 JSON / Markdown endpoint：`/api/v1/audit/store-profile`。
- 输出默认 store、prototype store、未来 database store 需求。
- 明确当前 `safeForCurrentRuntime=true`，但 `readyForProductionAudit=false`。
- 把生产审计阻塞项显式列出，避免误判为生产 ready。

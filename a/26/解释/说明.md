# OrderOps Node V26 运行调试归档

## 本版目标

V26 新增本地 promotion archive verification：

- `GET /api/v1/ops/promotion-archive/verification` 输出 JSON verification。
- `GET /api/v1/ops/promotion-archive/verification?format=markdown` 输出 Markdown verification report。
- 重新复核 manifest digest 和 recomputed manifest digest 是否一致。
- 逐项复核 `archive-summary`、`latest-evidence`、`ledger-integrity` 三类 artifact。
- 复核 archive name、state、summary、next actions 是否匹配当前 bundle。
- Dashboard 新增 `Archive Verification` 和 `Verification Report` 入口。
- 本功能只读取 Node 本地 ledger，不触碰 Java / mini-kv。

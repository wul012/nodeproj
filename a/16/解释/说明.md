# OrderOps Node V16 运行调试归档

## 本版目标

V16 新增本地 ops checkpoint diff：

- `GET /api/v1/ops/checkpoints/diff?baseId=...&targetId=...` 对比两个 checkpoint。
- 比较 readiness/runbook 决策摘要。
- 比较 summary signals。
- 比较 runbook totals。
- 比较 runbook step 状态变化。
- Dashboard 新增 `Diff Latest` 入口。
- 本接口只读取 Node 内存 checkpoint，不触碰 Java / mini-kv。

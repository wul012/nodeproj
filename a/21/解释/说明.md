# OrderOps Node V21 运行调试归档

## 本版目标

V21 新增本地 promotion evidence report：

- `GET /api/v1/ops/promotion-decisions/:decisionId/evidence` 输出 JSON 证据包。
- `GET /api/v1/ops/promotion-decisions/:decisionId/evidence?format=markdown` 输出 Markdown 证据包。
- 证据包汇总 decision、verification、digest、readiness/runbook/baseline 状态、原因数量和 next actions。
- Dashboard 新增 `Evidence Report` 入口，默认拉取最新 decision 的 Markdown evidence。
- 本功能只读取 Node 本地 ledger，不触碰 Java / mini-kv。

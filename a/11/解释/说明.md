# OrderOps Node V11 运行调试归档

## 本版目标

V11 新增本地 ops summary：

- 汇总 audit 请求统计。
- 汇总 operation intent 数量、状态、目标、风险。
- 汇总 dispatch ledger 数量和状态。
- 汇总 timeline event 数量和类型。
- 汇总 mutation rate limit 信号。
- Dashboard 新增 ops 指标卡。

本版只读 Node 内存状态，不访问 Java / mini-kv。

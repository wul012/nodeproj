# Node v93 说明：Workflow evidence verification endpoint

## 本版目标

v93 按 `docs/plans/v92-production-ops-roadmap.md` 推进，新增只读 workflow evidence verification endpoint。

它把 `.github/workflows/node-evidence.yml` 的安全边界变成可查询证据，避免 CI workflow 后续被改成带 secret、部署命令或上游写操作。

## 本版新增能力

- 新增 workflow evidence verification service。
- 新增 JSON / Markdown endpoint。
- 检查 `npm ci`、typecheck、test、build、safe HTTP smoke 是否存在。
- 检查 `UPSTREAM_PROBES_ENABLED=false`、`UPSTREAM_ACTIONS_ENABLED=false`。
- 检查没有 `secrets.`、`kubectl`、`docker push`、`scp`。
- 明确 `readOnly=true`、`executionAllowed=false`。

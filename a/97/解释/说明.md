# Node v97 说明：Audit store env config profile

## 本版目标

v97 按 `docs/plans/v95-production-gap-roadmap.md` 推进，新增 audit store 环境配置 profile。

本版只定义和验证 `AUDIT_STORE_KIND`、`AUDIT_STORE_PATH`、`AUDIT_STORE_URL` 的生产化配置边界，不真正切换运行时 audit store，不连接数据库，不迁移审计数据。

## 本版新增能力

- 新增配置项：
  - `AUDIT_STORE_KIND`
  - `AUDIT_STORE_PATH`
  - `AUDIT_STORE_URL`
- 新增 endpoint：`/api/v1/audit/store-config-profile`。
- 支持识别 `memory`、`file`、`database/postgres`。
- 对 database URL 做脱敏输出。
- 明确当前 runtime 仍是默认 in-memory audit store。

## 边界

本版没有启用 file-backed runtime，没有连接数据库，没有创建审计迁移，没有读取 secret，没有暴露真实数据库密码。

# OrderOps Node V27 运行调试归档

## 本版目标

V27 新增本地 promotion archive attestation：

- `GET /api/v1/ops/promotion-archive/attestation` 输出 JSON attestation。
- `GET /api/v1/ops/promotion-archive/attestation?format=markdown` 输出 Markdown attestation report。
- 把 V25 manifest digest、V26 verification digest、latest decision、checks 和 evidence sources 合成最终 `sealDigest`。
- 用 `handoffReady` 明确表示当前 archive 是否可以作为 promotion handoff 附件。
- Dashboard 新增 `Archive Attestation` 和 `Attestation Report` 入口。
- 本功能只读取 Node 本地内存状态，不触碰 Java / mini-kv。

const evidenceSources = input.manifest.artifacts.map((artifact) => {
  const verificationArtifact = input.verification.artifacts.find((candidate) => candidate.name === artifact.name);

  return {
    name: artifact.name,
    type: artifact.type,
    present: artifact.present,

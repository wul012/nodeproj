# v52 归档说明

## 版本主题

v52 新增统一上游观察台第一版：Node 通过 `/api/v1/upstreams/overview` 把 Java order platform 和 mini-kv 的只读状态汇总成统一观察对象。

本版工作量控制在一个小闭环：

```text
OpsSnapshotService live probes -> upstream overview -> dashboard/debug API
```

本版不修改 Java 高并发项目源码，不修改 mini-kv 源码。真实联调时只临时启动两个上游，验证后已停止。

## 关键代码

- `src/clients/miniKvClient.ts`
  - 新增 `MiniKvStatsJsonResult`
  - 新增 `health()`

- `src/routes/statusRoutes.ts`

- `src/ui/dashboard.ts`

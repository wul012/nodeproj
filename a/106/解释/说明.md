# Node v106 归档说明

## 本版目标

本版完成 file audit restart evidence report，用 file-backed audit runtime 验证审计事件可以被重新加载。

## 新增能力

- 新增 `/api/v1/audit/file-restart-evidence`
- 支持 JSON / Markdown
- file runtime 下执行恢复演练：
  - 读取写入前事件
  - 写入一条合成 audit event
  - 新建 `FileBackedAuditStore`
  - 重新加载同一 JSONL
  - 比较 digest
- memory runtime 下明确 `recoveryVerified=false`

## 安全边界

- 不接数据库
- 不做轮转、备份、加密
- 不启用生产审计
- `readyForProductionAudit=false`
- 不执行 Java replay
- 不执行 mini-kv 写命令

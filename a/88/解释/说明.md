# Node v88 说明：CI evidence command profile

## 本版目标

v88 按 `docs/plans/v86-production-readiness-roadmap.md` 推进，新增面向 CI 的只读 evidence command profile。

它把 typecheck、test、build、safe smoke、release evidence readiness gate 的推荐命令和安全环境变量标准化。

## 本版新增能力

- 新增 CI evidence profile service。
- 新增 JSON / Markdown endpoint。
- 明确安全环境变量：`UPSTREAM_PROBES_ENABLED=false`、`UPSTREAM_ACTIONS_ENABLED=false`。
- 标注哪些命令可在 CI 跑，哪些命令必须人工授权。
- 不创建 GitHub Actions 工作流，不写入 secret。

# Node v105 归档说明

## 本版目标

本版完成 operator identity contract rehearsal，把 header-derived operator identity 固定成可测试、可审计的生产雏形契约。

## 新增能力

- 新增 `/api/v1/security/operator-identity-contract`
- 支持 JSON / Markdown 输出
- `AuditEvent` 新增 `operatorIdentity`
- `accessGuard` 和 `operatorIdentity` 会写入同一条 audit event
- 身份解析覆盖：
  - `operatorId`
  - `roles`
  - `rawRoles`
  - `rejectedRoles`
  - `authSource`

## 安全边界

- header identity 只作为 rehearsal contract
- 不读取 secret
- 不实现 JWT
- 不接外部身份系统
- 不启用真实 enforcement
- `readyForProductionAuth=false`
- 不执行 Java replay
- 不执行 mini-kv 写命令

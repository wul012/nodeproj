# v53 归档说明

## 版本主题

v53 接入 Java v36 的订单平台只读运行概览：Node 在 `/api/v1/upstreams/overview` 中同时展示 Java health 和 Java `/api/v1/ops/overview` 的业务信号。

本版工作量控制在一个小闭环：

```text
Java health -> Java ops overview -> Node upstream overview business signals
```

本版不新增 Java 写操作，不修改 Java 项目源码，不修改 mini-kv 项目源码。

## 关键代码

- `src/clients/orderPlatformClient.ts`
  - 新增 `OrderPlatformOpsOverview`
  - 新增 `opsOverview()`
- `src/services/opsSnapshotService.ts`
  - Java 探测在 health 成功后读取 `/api/v1/ops/overview`
  - overview 可用时 Java 状态为 `online`
  - health 可用但 overview 不可用时 Java 状态为 `degraded`
  - overview 请求失败不会让 Node `/api/v1/upstreams/overview` 整体失败
- `src/services/upstreamOverview.ts`
  - Java observation 增加 `ops business overview`
  - Java readSignals 增加 `GET /api/v1/ops/overview`
  - Java signals 增加 `businessOverviewAvailable`
  - Java signals 增加 `application`、`orders`、`inventory`、`outbox`、`failedEvents`
- `test/opsSnapshotService.test.ts`
  - 覆盖 Java overview 可用路径
  - 覆盖 Java overview 不可用 fallback 路径
- `test/upstreamOverview.test.ts`
  - 覆盖 Java business overview 信号提取
- `README.md`
  - 更新 upstream overview 能力说明
- `代码讲解记录/57-java-ops-overview-v53.md`
  - 新增本版代码讲解

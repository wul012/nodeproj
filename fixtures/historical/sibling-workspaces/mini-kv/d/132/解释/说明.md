# mini-kv v132 命令结果归档说明

本版读取 Node v299 决策记录计划后，补充 `credential_resolver_runtime_shell_decision_record_non_participation_receipt`。mini-kv 只回显 Node v299 blocked decision record 和 Node v298 upstream echo verification，不实现、不启用、不调用 runtime shell，不读取凭证值，不解析 raw endpoint，不发起 HTTP/TCP，不写 ledger/schema/storage，不执行 LOAD/COMPACT/RESTORE/SETNXEX，也不成为 audit/order authority。

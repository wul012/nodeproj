# mini-kv v120 运行归档说明

## 版本目标

本版新增 credential resolver disabled implementation candidate 非参与回执，供 Node v274 在 Node v273 disabled implementation candidate review 之后只读验证。mini-kv 只回显 Node v273 的 disabled interface / fake wiring review 边界，不实现 resolver，不实例化 resolver client 或 secret provider，不读取 credential value，不解析 raw endpoint URL，不发送 external request，不连接 managed audit，不 auto-start，不写 storage / approval ledger / managed audit state / production record，也不执行 schema migration、LOAD、COMPACT、RESTORE 或 SETNXEX。

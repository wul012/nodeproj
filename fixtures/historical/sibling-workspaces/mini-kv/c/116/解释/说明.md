# mini-kv v116 运行归档说明

## 版本目标

本版新增 test-only resolver shell 非参与回执，供 Node v265 在 Node v264 credential resolver test-only shell contract 之后只读验证。mini-kv 只回显上游合同形状和边界，不实例化 resolver client 或 secret provider，不执行 fake resolver probe，不读取 credential value，不解析 raw endpoint URL，不发送外部请求，不连接 managed audit，不 auto-start，不写 storage 或 production record，不执行 schema migration、LOAD、COMPACT、RESTORE 或 SETNXEX，也不成为 managed audit storage backend。

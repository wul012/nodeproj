# mini-kv v101 command archive

## Scope

Version 101 follows the Node plan files read before implementation:

- `D:\nodeproj\orderops-node\docs\plans\README.md`
- `D:\nodeproj\orderops-node\docs\plans\v236-post-dry-run-envelope-roadmap.md`

The selected mini-kv scope is the runtime no-start / no-write evidence follow-up for the Node v236 dry-run request envelope and the Node v237 manual sandbox connection readiness gate. The change is evidence-only: it does not make mini-kv a managed audit storage backend, does not start mini-kv from Node or Java, does not read credential values, does not run schema rehearsal, and does not execute `LOAD`, `COMPACT`, `RESTORE`, or `SETNXEX`.

# mini-kv v117 运行归档说明

## 版本目标

本版新增 credential resolver fake-shell archive 非参与回执，供 Node v267 在 Node v266 archive verification 之后只读核验。mini-kv 只回显 Node v266 对 v264/v265 fake-shell 证据链的归档事实，不读取 Node 归档文件，不重跑 fake resolver shell 行为，不实例化 resolver client 或 secret provider，不读取 credential value，不解析 raw endpoint URL，不发送外部请求，不连接 managed audit，不 auto-start，不写 storage 或 production record，不执行 schema migration、LOAD、COMPACT、RESTORE 或 SETNXEX。

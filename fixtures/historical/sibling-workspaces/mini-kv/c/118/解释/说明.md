# mini-kv v118 运行归档说明

## 版本目标

本版新增 credential resolver production-readiness blocked-decision 非参与回执，供 Node v269 在 Node v268 blocked decision gate 之后只读验证。mini-kv 只回显 Node v268 的阻断决策和 Node v267 source echo 证据，不实现 resolver，不实例化 resolver client 或 secret provider，不读取 credential value，不解析 raw endpoint URL，不发送外部请求，不连接 managed audit，不 auto-start，不写 storage/approval ledger/managed audit state/production record，也不执行 schema migration、LOAD、COMPACT、RESTORE 或 SETNXEX。

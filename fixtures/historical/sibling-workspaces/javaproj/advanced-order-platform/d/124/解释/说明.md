拆分方式：

- `FailedEventSearchIntegrationTestSupport` 收口仓库清理、服务注入、`failedEventMessage` fixture 和 bad request 断言。
- `FailedEventSearchIntegrationTests` 保留 failed message search 与 replay attempt search。
- `FailedEventReplayApprovalSearchIntegrationTests` 承接 replay approval request/review/search/export/replay gate 场景。
- `FailedEventManagementSearchIntegrationTests` 承接 management status、management history、CSV export 场景。
- `FailedEventSearchValidationIntegrationTests` 承接 search/export/management validation 边界。

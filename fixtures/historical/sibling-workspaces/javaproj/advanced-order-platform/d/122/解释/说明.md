# Java v122 运行说明：Integration Tests 第一刀拆分

## 本版目标

按 Node `plans2/v287-post-test-only-fake-harness-precheck-roadmap.md` 的推荐，Java v122 只做 Integration Tests 第一刀拆分。

本版不改生产代码，不新增真实 fake harness runtime，不读取 credential value，不解析 raw endpoint URL，不打开 managed audit connection，不写 ledger，不执行 SQL。

## 拆分结果

原文件：

```text
src/test/java/com/codexdemo/orderplatform/OpsReleaseApprovalRehearsalLiveAggregationIntegrationTests.java
```


拆分方式：

- `OpsReleaseApprovalRehearsalLiveAggregationIntegrationTestSupport` 统一 `MockMvc`、失败事件数据清理、live aggregation replay fixture。
- 主测试保留 release approval rehearsal live aggregation 主合同和 failure taxonomy 覆盖。
- 新测试承接 `verificationHint`、`releaseApprovalInputs`、`liveSignals`、`executionBoundaries`、blockers 和 next actions 尾段 JSON contract。

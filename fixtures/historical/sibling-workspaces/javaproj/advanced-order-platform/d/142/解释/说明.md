## 本次范围

Java v142 消费 Node v306 的 approval prerequisite artifact intake plan，新增只读 echo receipt。它只证明 Java 已收到并回显 Node 定义的非 secret artifact contract：12 个 required fields、8 个 prohibited fields、9 个 rejection reasons、12 个 no-go boundaries。

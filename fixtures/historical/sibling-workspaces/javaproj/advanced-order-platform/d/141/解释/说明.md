# Java v141 运行说明

## 本次范围

Java v141 消费 Node v304 的 runtime shell chain stop-or-prerequisite decision record，新增只读 echo receipt。它只证明 Java 已收到并回显 Node 的阻塞决策：显式审批前置条件未满足，因此 runtime shell 链路继续 blocked。

本版不实现 runtime shell，不读 credential，不解析 raw endpoint URL，不创建 provider/client，不发外部 managed audit 请求，不写 ledger，不执行 SQL，也不要求 Docker。

全量测试通过。没有启动 Docker Desktop；全量测试中 Testcontainers 对 Docker 不可用有 warning，但未失败。没有启动或遗留 Java/Node/mini-kv 后台服务。

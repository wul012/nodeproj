# v91 运行说明

## 1. 版本目标

Java v91 按 Node `v234-post-blocked-execution-rehearsal-roadmap.md` 新增 `managedAuditSandboxConnectionPreconditionReceipt`。

它只读列出真实 managed audit sandbox connection 前必须具备的材料：owner approval artifact、credential handle review、schema rehearsal evidence、rollback path、timeout budget 和 manual abort marker。

本版不打开连接，不读取 credential value，不执行 SQL，不写 approval ledger，不启动 Java / mini-kv / 外部 audit 服务。

## 4. 边界

- Java 不创建 approval decision
- Java 不写 approval ledger
- Java 不持久化 approval record
- Java 不写 managed audit store
- Java 不读取 credential value
- Java 不执行 schema migration SQL
- Java 不打开 managed audit connection
- Java 不触发 deployment / rollback / restore
- Java 不自动启动外部服务

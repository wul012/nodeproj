# v92 运行说明

## 1. 版本目标

Java v92 按 Node `v236-post-dry-run-envelope-roadmap.md` 新增 `managedAuditSandboxConnectionDryRunEnvelopeEchoReceipt`。

它只读回显 Node v236 dry-run request envelope 的 6 个字段名：

```text
ORDEROPS_MANAGED_AUDIT_OWNER_APPROVAL_ARTIFACT_ID
ORDEROPS_MANAGED_AUDIT_SANDBOX_CREDENTIAL_HANDLE
ORDEROPS_MANAGED_AUDIT_SCHEMA_REHEARSAL_ID
ORDEROPS_MANAGED_AUDIT_ROLLBACK_PATH_ID
timeoutBudgetMs
ORDEROPS_MANAGED_AUDIT_MANUAL_ABORT
```

本版不打开 managed audit 连接，不读取 credential value，不执行 SQL，不写 approval ledger，不启动 Java / mini-kv / 外部 audit 服务。

## 3. 新 receipt 状态

顶层新增：

```text
managedAuditSandboxConnectionDryRunEnvelopeEchoReceipt
```

核心状态：

```text
consumedByNodeDryRunRequestEnvelopeVersion=Node v236
nextNodeReadinessGateVersion=Node v237
allEnvelopeFieldsEchoed=true
credentialValueExcluded=true
readyForManagedAuditSandboxAdapterConnection=false
readyForProductionAudit=false
```

完整 header 场景下 `readyForNodeV237ManualSandboxConnectionReadinessGate=true`，表示 Node v237 可以消费该只读 echo receipt；默认缺 header 场景下该字段为 false。

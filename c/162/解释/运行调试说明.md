# v162 运行调试说明：cross-project release verification intake gate

## 本版目标

v162 按最新全局计划消费两个已完成上游版本：

```text
Java v54：release verification manifest
mini-kv v63：release verification manifest
```

Node 本版生成跨项目 release verification intake gate，只读取和固化两边已归档的 release 证据，不启动 Java / mini-kv，不执行 Java Maven，不执行 mini-kv CMake/CTest，也不触发任何上游写操作。

## 安全边界

本版 HTTP smoke 和截图只启动 Node 本地服务，验证后已停止。

运行环境保持：

```text
UPSTREAM_PROBES_ENABLED=false
UPSTREAM_ACTIONS_ENABLED=false
```

本版没有启动、停止、构建、测试或修改 Java / mini-kv。Java v54 和 mini-kv v63 已提前完成，本版只读核对它们的 tag、manifest、归档说明和 release 边界语义。

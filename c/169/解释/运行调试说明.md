## 安全边界

本版 HTTP smoke 和截图只启动 Node 本地服务，验证后停止。

运行环境保持：

```text
UPSTREAM_PROBES_ENABLED=false
UPSTREAM_ACTIONS_ENABLED=false
```

本版没有启动、停止、构建、测试或修改 Java / mini-kv。

Node v169 明确不允许：

```text

Authorize production operations
Read production secret values
Connect to production database
Execute mini-kv LOAD / COMPACT / SETNXEX
Bypass v167/v168 evidence chain
Continue appending overlapping versions to the v166 plan
```

# v164 运行调试说明：release report shared helpers refactor

## 本版目标

v164 是一个插入的 Node 内部重构版，用来回应最近报告服务膨胀的问题。

本版不推进原计划中的 cross-project release bundle gate，因为 Java v56 / mini-kv v65 仍应由其他窗口并行推进。本版只整理 Node 现有 v162 / v163 release report 的重复代码：

```text
Markdown 报告拼装
blocking message 追加
step rendering
forbidden operation rendering
digest helper
aggregate ready check
```

## 安全边界

本版 HTTP smoke 和截图只启动 Node 本地服务，验证后已停止。

运行环境保持：

```text
UPSTREAM_PROBES_ENABLED=false
UPSTREAM_ACTIONS_ENABLED=false
```

本版没有启动、停止、构建、测试或修改 Java / mini-kv。其他窗口可以继续推进 Java v56 和 mini-kv v65。

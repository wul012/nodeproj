人工窗口顺序

  1. 记录关闭窗口 baseline：rehearsalState=closed-skipped。
  2. 操作者手动启动 Java 和 mini-kv，Node 不启动上游。
  3. Node 仅以 UPSTREAM_PROBES_ENABLED=true、UPSTREAM_ACTIONS_ENABLED=false 采集只读结果。
  4. 操作者手动停止 Java 和 mini-kv。
  5. 归档 JSON、Markdown、digest、runtime env 和窗口结果。

只读边界

  • Java：GET /actuator/health、GET /api/v1/ops/overview
  • mini-kv：HEALTH、INFOJSON、STATSJSON
  • 禁止：自动启动上游、打开 actions、Java 写接口、mini-kv 写入或管理命令。

# v161 运行调试说明：controlled idempotency drill runbook

## 本版目标

v161 按最新计划消费两个已完成上游版本：

```text
Java v53：idempotency store abstraction
mini-kv v62：ttl recovery evidence
```

Node 本版生成 controlled idempotency drill runbook，只做人工 dry-run 说明和证据汇总，不启动 Java / mini-kv，不执行 Java 写接口，不执行 mini-kv `SETNXEX`。

## 安全边界

本版 HTTP smoke 和截图只启动 Node 本地服务，验证后已停止。

运行环境保持：

```text
UPSTREAM_PROBES_ENABLED=false
UPSTREAM_ACTIONS_ENABLED=false
```

本版没有启动、停止、构建、测试或修改 Java / mini-kv。

Java v53 和 mini-kv v62 已提前完成，本版只读取它们的 tag 和 evidence 语义作为 runbook 输入。

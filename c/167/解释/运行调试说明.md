## 本版目标

v167 按最新全局计划推进 Node 自己负责的版本：

```text
Node v167：rollback execution preflight contract
```

前置条件已经只读核对完成：

```text
Java v58：production rollback SQL review gate sample 已完成并推送
```

## 安全边界

本版 HTTP smoke 和截图只启动 Node 本地服务，验证后已停止。

运行环境保持：

```text
UPSTREAM_PROBES_ENABLED=false
UPSTREAM_ACTIONS_ENABLED=false
```

本版没有启动、停止、构建、测试或修改 Java / mini-kv。

Node v167 明确不允许：

```text
Execute rollback SQL from Node
Connect to a production database
Trigger Java rollback from Node
Execute mini-kv LOAD / COMPACT / SETNXEX
Execute mini-kv restore or admin commands
Treat mini-kv restore package as Java order authority
Treat preflight contract as production rollback approval
```

## 一句话结论

v167 把 Java v58 的 SQL review gate 与 mini-kv v67 的 restore dry-run operator package 合成了只读 preflight contract，让回滚执行前的人工证据更接近生产流程，但仍不授权真实 SQL、恢复、回滚或上游动作。

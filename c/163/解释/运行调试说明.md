# v163 运行调试说明：release rollback readiness runbook

## 本版目标

v163 按最新全局计划消费两个已完成上游版本：

```text
Java v55：deployment rollback evidence sample
mini-kv v64：runtime artifact rollback evidence sample
```

Node 本版生成 release rollback readiness runbook，只做人工 dry-run 回滚窗口准备说明和证据汇总，不启动 Java / mini-kv，不执行 Java rollback，不执行 mini-kv `LOAD` / `COMPACT` / `SETNXEX`，不打开生产写开关。

## 安全边界

本版 HTTP smoke 和截图只启动 Node 本地服务，验证后已停止。

运行环境保持：

```text
UPSTREAM_PROBES_ENABLED=false
UPSTREAM_ACTIONS_ENABLED=false
```

本版没有启动、停止、构建、测试或修改 Java / mini-kv。Java v55 和 mini-kv v64 已提前完成，本版只读核对它们的 tag、evidence、归档说明和回滚边界语义。

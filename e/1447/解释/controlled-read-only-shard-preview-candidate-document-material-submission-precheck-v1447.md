# v1447 candidate document material submission precheck

v1447 starts the submission precheck layer with the material-source-package checkpoint. It consumes the v1446 material request package and requires the source precheck, reviewed document envelope, and field-completeness request material before any later submission can be considered.

The checkpoint is intentionally read-only. It records the source request item and acceptance check coverage, keeps real document and payload counts at zero, and keeps candidate document material submission and intake disabled.

tagged, the final batch verification and Node Evidence run are green, and a fresh
census proves every claimed baseline deletion. A 9/10 recommendation requires:

1. zero runtime import cycles;
2. removal of the v114-v117 mini-kv builder quartet from both long-function and
   complexity ledgers, with v117's near-limit service removed from that ledger;

   no weakened tests, and no expanded execution authority.

The score remains below 9 if these targets pass only by moving complexity, hiding
it below the scanner, relaxing thresholds, or replacing exact contracts with prose.
